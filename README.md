# Assembler Gif & Memes

A Memes and Gifs web page in which users can upload their files and share it with everyone.


## Authors

- [Marcel Barreiro](https://github.com/marcelbbarreiro)


## 🔐 Requirements

 - Home page that will show the content uploaded by users
 - Main Navbar that will include: Content search bar:
 - Upload content page: If the user is logged in, a multimedia resource can be uploaded from a an external link

## 🛠 Technologies
React, HTML, Tailwind, Firebase, Toast...
